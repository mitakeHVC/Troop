モックアップ概要

BOPIS機能の包括的なUIモックアップ

BOPIS（Buy Online, Pick up In Store）機能の画面モックアップを一覧で表示しています。各ユーザータイプ（顧客、ピッカー、受付担当、テナント管理者）向けの画面を確認できます。

実際の画面を確認するには、以下のリンクをクリックしてください。

顧客向け画面

オンライン注文から店舗受取までの顧客体験

画面名 説明 アクション
ログイン画面 顧客がシステムにログインするための画面 表示
テナント選択画面 利用するテナント（イベント）を選択する画面 表示
商品一覧画面 テナントの商品一覧を表示する画面 表示
商品詳細画面 商品の詳細情報を表示する画面 表示
カート画面 カートに追加した商品を確認する画面 表示
チェックアウト画面 注文を確定するための画面 表示
注文確認画面 注文完了後のQRコード表示画面 表示
注文履歴画面 過去の注文履歴を確認する画面 表示

ピッカー向け画面

商品ピッキングと準備管理

画面名 説明 アクション
ログイン画面 ピッカーがシステムにログインするための画面 表示
ダッシュボード画面 ピッカーのメイン画面、注文一覧を表示 表示
注文詳細画面 ピッキングする注文の詳細を表示する画面 表示
準備完了画面 商品準備完了を確認する画面 表示

受付担当向け画面

顧客対応と商品受け渡し管理

画面名 説明 アクション
ログイン画面 受付担当がシステムにログインするための画面 表示
ダッシュボード画面 受付担当のメイン画面、準備完了注文一覧を表示 表示
本人確認画面 顧客の本人確認を行うための画面 表示
受け渡し完了画面 商品受け渡し完了を確認する画面 表示

テナント管理者向け画面

システム全体の管理と設定

画面名 説明 アクション
ログイン画面 テナント管理者がシステムにログインするための画面 表示
ダッシュボード画面 テナント管理者のメイン画面、全体状況を表示 表示
商品管理画面 商品の追加・編集・削除を行う画面 表示
スタッフ管理画面 スタッフの追加・編集・削除を行う画面 表示
レーン管理画面 レーンの追加・編集・削除を行う画面 表示

使用方法

モックアップの操作ガイド

基本操作

  • 各ユーザータイプのログイン画面からフローを開始
  • 画面上のボタンやリンクで画面間を移動
  • ダークモード切り替えボタンでテーマ変更

注意事項

  • このモックアップは機能概要の表示のみ
  • 実際の動作やデータ処理は行いません
  • 実装時の参考資料として活用してください
開発者向け: 各画面のHTMLファイルには、実装時に参考となるUI構造とスタイリングが含まれています。
