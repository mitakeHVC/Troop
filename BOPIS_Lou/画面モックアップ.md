# BOPIS機能 画面モックアップ計画書

**作成日**: 2025/05/30
**作成者**: PdMチーム
**バージョン**: 0.1（初期ドラフト）
**ステータス**: Draft

このドキュメントでは、BOPIS機能の主要画面のHTMLモックアップを作成するための計画を提供します。実際のHTMLファイルはCodeモードで作成します。

## 1. ディレクトリ構造

```
docs/BOPIS_Lou/mockups/
├── styles.css                # 共通スタイルシート
├── images/                   # 画像ファイル
│   ├── tenant1.jpg
│   ├── tenant2.jpg
│   ├── tenant3.jpg
│   ├── product1.jpg
│   ├── product2.jpg
│   ├── product3.jpg
│   └── qrcode.png
├── customer/                 # 顧客向け画面
│   ├── login.html            # ログイン画面
│   ├── register.html         # 登録画面
│   ├── tenants.html          # テナント選択画面
│   ├── products.html         # 商品一覧画面
│   ├── product-detail.html   # 商品詳細画面
│   ├── cart.html             # カート画面
│   ├── checkout.html         # チェックアウト画面
│   ├── order-confirmation.html # 注文確認画面（QRコード表示）
│   └── orders.html           # 注文履歴画面
├── picker/                   # ピッカー向け画面
│   ├── login.html            # ログイン画面
│   ├── dashboard.html        # ピッカーダッシュボード
│   ├── order-detail.html     # 注文詳細画面
│   └── complete.html         # 準備完了画面
├── counter/                  # 受付担当向け画面
│   ├── login.html            # ログイン画面

│   ├── verify.html           # 本人確認画面

    ├── login.html            # ログイン画面

```

## 2. 共通スタイル

共通のスタイルシート（styles.css）には以下の要素を含めます：

- 基本スタイル（フォント、色、マージン、パディングなど）
- ヘッダーとフッター
- ボタンスタイル（プライマリ、セカンダリ、成功、危険など）
- カードコンポーネント
- フォーム要素
- アラートメッセージ
- テーブルスタイル
- レスポンシブデザイン用のメディアクエリ

## 3. 画面詳細

### 3.1 顧客向け画面

#### 3.1.1 ログイン/登録画面
- メールアドレスとパスワード入力フォーム
- ログインボタン
- 新規登録へのリンク

#### 3.1.2 テナント選択画面
- 利用可能なテナント（イベント）のカードリスト
- 各カードには画像、イベント名、日付、会場情報を表示
- 選択ボタン

#### 3.1.3 商品一覧画面
- 検索バーとフィルター
- 商品カードのグリッド表示
- 各カードには画像、商品名、価格、詳細ボタン、カートに追加ボタンを表示

#### 3.1.4 商品詳細画面
- 大きな商品画像
- 商品名、価格、説明
- サイズや数量の選択オプション
- カートに追加ボタン

#### 3.1.5 カート画面
- カート内の商品リスト（画像、名前、価格、数量）
- 数量変更と削除オプション
- 小計と合計金額
- チェックアウトボタン

#### 3.1.6 チェックアウト画面
- 注文内容の確認
- 受取時間枠の選択
- 支払い情報入力フォーム
- 注文確定ボタン

#### 3.1.7 注文確認画面（QRコード表示）
- 注文成功メッセージ
- 注文番号と受取時間枠
- QRコード表示
- QRコードの保存・共有オプション
- 注文内容の詳細

#### 3.1.8 注文履歴画面
- 過去の注文リスト
- 各注文の詳細（注文番号、日付、金額、ステータス）
- QRコード再表示オプション

### 3.2 ピッカー向け画面

#### 3.2.1 ログイン画面
- スタッフID/メールアドレスとパスワード入力フォーム
- ログインボタン

#### 3.2.2 ピッカーダッシュボード
- 新規注文通知エリア
- 処理中の注文リスト
- 完了した注文リスト
- ステータス更新オプション

#### 3.2.3 注文詳細画面
- 注文情報（注文番号、顧客名、受取時間枠）
- ピッキングリスト（商品名、画像、数量、場所）
- ステータス更新ボタン（準備開始、準備完了）

#### 3.2.4 準備完了画面
- 準備完了確認メッセージ
- 注文情報の要約
- 受付担当への引き渡し確認
- 次の注文へのリンク

### 3.3 受付担当向け画面

#### 3.3.1 ログイン画面
- スタッフID/メールアドレスとパスワード入力フォーム
- ログインボタン

#### 3.3.2 受付担当ダッシュボード
- 準備完了通知エリア
- 待機中の顧客リスト
- 処理中の注文リスト
- 完了した注文リスト
- レーンステータス表示

#### 3.3.3 本人確認画面
- QRコードスキャン結果
- 顧客情報（名前）
- 注文内容（商品リスト）
- 本人確認質問（ランダムな商品の購入点数確認）
- 確認完了ボタン

#### 3.3.4 受け渡し完了画面
- 受け渡し完了確認メッセージ
- 注文情報の要約
- 次の顧客へのリンク

### 3.4 テナント管理者向け画面

#### 3.4.1 ログイン画面
- メールアドレスとパスワード入力フォーム
- ログインボタン

#### 3.4.2 ダッシュボード
- 注文統計（総数、完了、処理中）
- 売上グラフ
- 人気商品ランキング
- スタッフ稼働状況
- レーン稼働状況

#### 3.4.3 商品管理画面
- 商品リスト（画像、名前、価格、在庫数）
- 商品追加・編集・削除機能
- 在庫管理機能

#### 3.4.4 スタッフ管理画面
- スタッフリスト（名前、役割、ステータス）
- スタッフ追加・編集・削除機能
- 役割割り当て機能（ピッカー、受付担当）

#### 3.4.5 レーン管理画面
- レーンリスト（名前、ステータス、担当者、待ち人数）
- レーン追加・編集・削除機能
- レーンステータス変更機能
- 受付担当者割り当て機能
