# BOPIS機能モックアップ

BOPIS（Buy Online, Pick up In Store）機能のモックアップです。このモックアップは、ヒアリングシートに基づいて、ピッカーと受付担当の役割分担、QRコード認証時のピッカー通知、レーン振り分け、本人確認プロセスなどの新しいフローを実装しています。

## 機能概要

1. **顧客向け機能**：
   - オンラインでの商品選択から注文、QRコード取得までの一連のフロー
   - 受取時間枠の選択機能（混雑状況の可視化）
   - QRコードによる注文確認システム

2. **ピッカー向け機能**：
   - 注文通知の受信
   - 商品のピッキングと準備
   - 準備完了通知の送信
   - オフライン対応機能

3. **受付担当向け機能**：
   - QRコードスキャン機能
   - 本人確認プロセス（名前と商品点数の確認）
   - 商品受け渡し完了処理
   - オフライン対応機能

4. **テナント管理者向け機能**：
   - 全体状況の把握（ダッシュボード）
   - 商品管理（在庫管理含む）
   - スタッフ管理（役割割り当て含む）
   - レーン管理（レーン割り当てルール設定含む）
   - リアルタイム在庫管理機能

    ├── hearing-sheet-Lou20250526.md       # 初期ヒアリングシート

## 使用方法

1. モックアップを表示するには、`BOPIS_Lou/mockups/index.html`をブラウザで開いてください。
2. インデックスページから各画面にアクセスできます。
